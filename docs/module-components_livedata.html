Module: components/livedata

Files module responsible for AWS SNS, S3, ElasticTranscoder Handles file uploads, transcoding, download Depends on AWS S3, Must supply Credentials as a JSON in constants.AWS_CREDS It will generate buckets under sflix_{USER_ID} Create the following notifications for SNS and supply the configuration sflix_notification_progressing sflix_notification_completed sflix_notification_warning sflix_notification_error
